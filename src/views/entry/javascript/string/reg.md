## 正则表达式
用来匹配字符串的一种规则对象。由两个部分组成：元字符与修饰符。
先来了解一下元字符：

+ 量词元字符

```javascript
/** 量词元字符用来设置出现的次数：
 *   *  代表0到多次
 *   +  代表1到多次
 *   ?  代表0或者1次
 *   {n}  代表出现n次
 *   {n,} 代表出现n到多次
 *   {n, m}  代表出现n-m次
*/
```

+ 特殊元字符

```javascript
/** 单个或组合在一起有特殊含义
 *   \  转义字符
 *   .  除了\n(回车)以外的所有字符
 *   ^  以哪一个字符串作为开始
 *   $  以哪一个字符串作为结束
 *   \n  换行
 *   \d  数字  \D  非数字
 *   \s  空白字符  \S  非空白字符
 *   x|y  x 或者 y中的一个
*/
```
